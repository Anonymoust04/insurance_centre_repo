===========================================================
  NOTES FOR DEVELOPER
===========================================================

  1. All images in /public/ are served from the root URL.
     Example: public/admin/logo-brand.png → accessed as /admin/logo-brand.png

  2. In Next.js static export, use <img> with src="/admin/logo-brand.png"
     or Next.js <Image> component with unoptimized={true} (already set in config).

  3. Recommended sizes are at 1x. Provide 2x (@2x) versions for retina screens
     by naming them: logo-brand@2x.png (optional but improves quality on mobile).

  4. All images with transparent backgrounds should be saved as PNG.
     Photos/illustrations can use WebP for smaller file size.

  5. The current build uses ONLY Tabler icon SVGs and emoji for all visuals.
     Swapping to real images is purely cosmetic — no code changes required
     except updating src="" attributes where applicable.
